Get Control Center

How it works: ROG Control Center is a graphical interface built on top of two powerful CLI tools‚Äîasusctl and supergfxctl. These daemons handle all hardware communication, while the GUI gives you an intuitive way to control everything.

asusctl manages fans, LEDs, keyboard backlight, battery charge limits, and device-specific features. supergfxctl handles GPU mode switching (iGPU/Hybrid/dGPU) and MUX control.

Why both? New users get a polished app. Power users and server admins can script everything via CLI. Both paths use the same battle-tested foundations.
