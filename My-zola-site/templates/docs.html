ROG Control Center is an open-source Linux utility that gives you full control over your ASUS ROG laptop's performance settings, fan curves, RGB lighting, and power management.

Installation

Install ROG Control Center using your preferred package manager or build from source.

Using Package Manager

sudo pacman -S rog-control-center
sudo systemctl enable --now rog-control-center

Build from Source

git clone https://github.com/rog-control/rog-control-center
cd rog-control-center
cargo build --release
sudo make install
âœ“ Verify Installation: Run rogcc --version to confirm successful installation.

Basic Usage

After installation, you can control your laptop through the GUI application or command line interface.

Launch GUI

rog-control-center-gui

Command Line Examples

# Switch to performance mode
rogcc profile set performance

# Adjust fan curve
rogcc fan set-curve custom --temp 50:40 60:60 70:80

# Control RGB lighting
rogcc rgb set-mode rainbow --speed 2

Configuration

ROG Control Center uses a YAML configuration file located at ~/.config/rog-control-center/config.yaml. You can customize default profiles, fan curves, and behavior.

default_profile: balanced
startup_delay: 5
notifications:
  profile_switch: true
  battery_threshold: 20
